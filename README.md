# elqueuebeta

## To run Back-End:

1. Install maven if necessary 

`sudo apt-get install maven`

2. Run build from root folder 
  
  `mvn clean package`

3. Run server 
  
  `java -jar target/elqueuebeta-0.0.1-SNAPSHOT.jar`

## To run Front-End:

1. Install node staff if necessary 
  
  `cd front`
  
  `npm install`
  
2. Run Front-End from "/front" folder 
  
  `cd front`
  
  `npm start`
